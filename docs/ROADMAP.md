# Development Roadmap

## Milestone 1 — Core Calculation Engine

- Integrate Swiss Ephemeris bindings.
- Implement timezone-safe datetime normalization.
- Validate planetary positions and house cusps against reference datasets.

## Milestone 2 — KP Charting Foundation

- Build Rasi, Bhava, KP cuspal, and Navamsa generation.
- Implement star-lord and sub-lord decomposition.
- Provide North/South chart renderers.

## Milestone 3 — Timing and Analysis

- Implement Vimshottari dasha hierarchy.
- Build transit comparison and trigger detection.
- Create event finder condition language.

## Milestone 4 — Practitioner Workflow

- Add horary chart flow.
- Add Muhurt selector.
- Add cue cards and printable reports.

## Milestone 5 — Data & Productization

- Add chart database, backup/restore.
- Add location resolver with PIN/ZIP support.
- Add chart adjustment (birth-time rectification) utilities.
